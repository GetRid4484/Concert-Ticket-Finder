# Concert Finder

A modern web application to find concerts near you and add them to your Google Calendar.

## How to Run Locally

1.  **Open in VS Code**:
    - Open Visual Studio Code.
    - Go to **File** > **Open Folder...**
    - Navigate to this folder: `concert-finder`

2.  **Install Dependencies** (if you haven't already):
    - Open the terminal in VS Code (`Ctrl` + `\``).
    - Run:
        ```bash
        npm install
        ```

3.  **Start the App**:
    - In the terminal, run:
        ```bash
        npm run dev
        ```
    - You will see a local URL (e.g., `http://localhost:5173`).
    - **Ctrl + Click** that link to open the app in your browser.

## Features

- **Location Search**: Find concerts by city or zip code.
- **Event Details**: View artist, venue, and time information.
- **Ticket Links**: Direct links to purchase tickets.
- **Calendar Integration**: One-click "Add to Calendar" button for Google Calendar.
